GIT COMMANDS

  1. git clone : use this command along with the link of the repository to download in your computer.
  2. touch : use this command along with a filename to create a file in the repository
  3. git add: use this command along with the filename to add file in the repository
  4. git commit -m "your_message": to describe the changes made in the file
  5. git status : to find the status of the repository
  6. git push : to add the updated repository into github
  7. git pull : to download the latest version of the code
  8. git log : to find the history of all the commits made in the repository
  9. git reset : to go to previous version of the repository
